# Project Summary

Project COSMOS is not a traditional portfolio.

It is a cinematic scrolling experience where visitors travel through one continuous universe.

The interface language is inspired by cybersecurity.

The environment is inspired by realistic astronomy.

Every chapter contributes to one connected story.

The visitor should never feel like they are navigating between separate pages.

---

# Project Vision

The goal is to create an Awwwards-quality interactive experience that combines:

• Software Engineering
• Cybersecurity
• Space Exploration
• Storytelling
• Motion Design

The experience should feel premium, calm, mysterious, and cinematic.

---

# Core Philosophy

Space tells the story.

Cybersecurity defines the interface.

Engineering drives the interaction.

Every animation exists to support storytelling.

Nothing exists purely for decoration.

# Animation Rules

Every animation must

✔ Support the story

✔ Be reversible

✔ Respect reduced motion

✔ Use transform + opacity whenever possible

✔ Remain smooth at 60 FPS

Never create chapter-specific animation systems.

Always reuse the shared animation infrastructure.
